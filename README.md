# UK, Portugal, Spain — Public Flood Data Sources

A catalogue of openly published flood-extent datasets across the United Kingdom (England, Wales, Scotland, Northern Ireland), Portugal (mainland, Madeira, Açores), and Spain (national, Confederaciones Hidrográficas, autonomous-community water agencies, Canary Islands). Each row is a dataset or service endpoint published by a national, devolved, or regional authority, plus EU-INSPIRE fallbacks and a labelled set of related-but-out-of-scope sources for completeness.

All sources listed are open data (OGL v3.0 in the UK, CC-BY-4.0 in Portugal and Spain).

**Coverage:** 28 UK · 17 Portugal · 35 Spain · 3 EU fallback · 11 related/out-of-scope · **94 total**

## Schema

| Column | Meaning |
|---|---|
| ID | Stable identifier `<COUNTRY>-<REGION>-<NNN>[a/b]` |
| Country | UK, PT, ES, EU, or OUT (out-of-scope-but-related) |
| Jurisdiction | Sub-national (England / Madeira / Cataluña / etc.) |
| Agency | Publishing authority |
| Dataset | Dataset / product name |
| Flood type | fluvial / coastal / surface water / reservoir / tsunami / historic |
| Return periods | Native AEP or T-year values published |
| Climate scenario | present-day / RCP4.5 / RCP8.5 / "futuro" / etc. |
| Formats | File formats and service types available |
| License | Open-data licence (OGL v3.0, CC-BY-4.0, etc.) |
| Delivery mode | download / server (WMS/WFS/REST) / hybrid |
| Primary URL | Landing page or canonical dataset URL |
| WMS URL | OGC Web Map Service endpoint, if any |
| WFS URL | OGC Web Feature Service endpoint, if any |
| Download URL | Direct file download or ATOM/REST endpoint |
| Notes | Semantics, gotchas, layer names, coverage notes |
